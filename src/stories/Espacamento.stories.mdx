# Espaçamento

O CSS nos permite espaçar elementos usando `padding` e `margin`,
além dos posicionamentos absolutos. O espaçamento muitas vezes se perde do design
da tela para a implementação, para resolver esse problema e ter uma padrão consistente
no sistema definiu-se esse fundamento do Design System. As definições aqui não estão
escritas em pedra, até porque um dos principios do nossso Design System é "Contexto acima
de consistência". 

<hr />
<br />

## Recomendações

-  Para definição das margens e espaçamentos recomendamos a utilização da escala linear
de razão 2 (padrão do Bootstrap), que já vem sendo utilizado pelos designers, com adicional de
mais opções de espaçamento além do definido do Bootstrap, seguindo a escala.

-  Sendo a margem negativa um padrão raramente utilizado no Cidade Saudável, não foram
definidos tokens. É passível o uso de margem negativa em componentes específicos, mas recomenda-se
que se use adequadamente como extrapolação de um elemento e não para compor/atingir um espaçamento
desejado.

-  Para melhor consistência e facilidade na manutenção de códigos deve-se evitar a utilização de 
espaçamentos combinados, como combinar alguns `margin` e `padding` de elementos para atingir
determinado espaçamento não definido na escala. Caso seja realmente necessário tal espaçamento
é preferível adicionar classe específica para o elemento em questão com o tal espaçamento.

- É preferível combinar posicionamentos (`esquerda`,`direita`,`superior` e `inferior`) com margens ou padding
menores em detrimento de usar uma margem ou padding muito grande e fora da faixa definida na escala.

## Escala de espaçamento
<br />

A escala utilizada é baseada no Bootstrap sendo os tamanhos um dos valores da escala abaixo.

export const Template = () => ({
    template:`<space />`,
});

<Story name="escala">
    { Template.bind({}) }
</Story>

Para usar os tokens de espaçamento utilize `@include padding($direcao, $tamanho);` ou `@include margin($direcao, $tamanho);`
onde `$direcao` é um dos valores:
- `esquerda`;
- `direita`;
- `superior`;
- `inferior`;
- `horizontal`;
- `vertical`;
- `onidirecional`.
