# StepperInput

### StepperInputs são inputs numéricos com botões de incremento e decremento sempre visíveis
---
<br />

## Recomendamos o uso de StepperInputs quando:
- Você preicsar de um input do tipo número
- Quando o número a ser adicionao ao StepperInput for pequeno ou de poucos dígitos

<br />

## Não recomendamos o uso de StepperInputs quando:
- O dado a utilizado com o componente não for numérico
- O dado a utilizado com o componente for muito grande
- O dado a utilizado com o componente não for incremental
