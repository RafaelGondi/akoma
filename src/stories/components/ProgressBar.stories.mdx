# ProgressBar

### ProgressBars são componentes utilizados para indicar o status e progresso de algo aos usuários.
---
<br />

## Recomendamos o uso de ProgressBrs quando:
- Você precisar transmitir a ideia do progresso de algo.
- Você precisar transmitir a ideia do status de algo.
- Você desejar chamar a atenção do usuário para uma informação específica de maneira gráfica.

<br />

## Não Recomendamos o uso de ProgressBrs quando:
- Para criar loaders.
- Você precisar de muita precisão ao mostrar os números ou progresso.
- Para criar steppers.
