# Popover

### Popovers são pop-ups que surgem quando o usuário clica em um determinado elemento.
### São usados para emular o comportamento de tooltips, e para mostrar conteúdos que estavam previamente ocultos.
---
<br />

## Recomendamos o uso de Popovers quando:
- Você desejar colocar filtros ou textos que serão apresentados apenas após a iteração do usuário com algum elemento da interface.

<br />

## Não recomendamos o uso de Popovers quando:
- Você precisar de um modal.
- Você estiver procurando uma solução de navegação.
- Utilizado junto a forms para registrar algo.
- Você desejar utilizar o componente com muito conteúdo
