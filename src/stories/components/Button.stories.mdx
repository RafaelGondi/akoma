# Button

### Button são componentes utilizados destacar palavras ou frases importantes em um determinado contexto.
---
<br />

## Recomendamos o uso de Buttons quando:
- Você deseja guiar a atenção do usuário para palavras ou frases espefícias na interface (nesses cenários Buttons animados podem ser ainda mais efetivos que os comuns)
- Você está implementando alguma feature de <i>diff</i> e precisa indicar elementos que foram substituídos por outros

<br />

## Não recomendamos o uso de Buttons quando:
- A palavra ou sentença se repetir na interface como em títulos de card, headers, etc. Para esses casos, use mecanismos de hierarquia tipográfica como peso da fonte, tamaho ou cor
- O elemento sobre do qual o Button será <i>wrapper</i> não for um elemento de texto.
- Forem destacados trechos muito grandes de texto. Isso pode acabar resultando no oposto do que se espera com o uso de Buttons
