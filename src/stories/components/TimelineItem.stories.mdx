# Timeline

### Timelines são componentes utilizados para apresentar informações em ordem cronológica.
---
<br />

## Quando usar:
- Se tem uma série de informações que podem ser categorizadas por data.

<br />

## Quando não usar:
- Se precisa de um componente de stepper.
- A ordem das informações que você está apresentando é irrelevante.

<br />

## Observações:
- A timeline espera um array de objetos com os campos: `date`, `title` e `text`, como espeficiado na tabela de props abaixo;
- Caso deseje que os pins da timeline tenham estilo com borda e sem preenchimento, você pode especificar a Propriedade
`_dotStyle` no objeto de configurações. Essa propriedade aceita dois valores: `hollowed` para estilo com borda e `filled`
para estilo com preenchimento.
