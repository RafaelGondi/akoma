# Badge

### Badge são indicadores de status utilizados para tornar evidentes metadados importates.
---
<br />

## Recomendamos o uso de Badges quando:
- Você quiser mostrar status associados com a lógica de negócio ou açoes do usuário.
- O conteúdo a ser mostrado for *readonly*
- For preciso categorizar algo

<br />

## Não recomendamos o uso de Badges quando:
- O conteúdo que a badge representa puder ser setado ou removido pelo usuário.
- O click no componente precisar executar uma ação ou funcionalidade.
