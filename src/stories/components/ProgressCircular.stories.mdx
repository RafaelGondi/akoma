# ProgressCircular

### ProgressCirculars são componentes quem indicam o status/progresso de uma operação em andamento, em formato circular.
---
<br />

## Recomendamos o uso de ProgressCirculars quando:
- Você precisa transmitir a ideia de progresso, percentual de uma operação em andamento.
- Você precisa transmitir a ideia de status de uma operação em andamento, que fração dela já foi concluída.
- Você deseja chamar a atenção do usuário para uma informação específica de maneira gráfica.

<br />

## Não Recomendamos o uso de ProgressCirculars quando:
- Você precisa representar fração de um processo com muitas etapas.
- Você precisa de um loader.
- Você precisar de dígitos de precisão decimal ao mostrar os números ou progresso.
- Você precisar representar steppers.
