# Wizard

### Wizards são componentes utilizados para exibir e organizar conteúdo, facilitando a comparação e análise de informações.
---
<br />

## Quando usar:
- Desejar comparar dados.
- For necessário exibir dados de forma tabulada.

<br />

## Quando não usar:
- For necessário interagir/realizar uma ação com o conteúdo apresentado na tabela.

<br />
