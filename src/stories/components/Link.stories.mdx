# Link

### Links são componentes usados ​​para navegar para outra página e podem ser abertos em uma nova guia.
---
<br />

## Recomendamos o uso de Links quando:
- For necessário navegar para outra seção ou página.

- For necessário redirecionar para um site ou documento externo.

<br />

## Não recomendamos o uso de Links quando:
- For necessário executar uma ação.
