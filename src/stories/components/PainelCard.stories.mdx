# PainelCard

### PainelCards são componentes utilizados para um estilo padrão de card com header com título, subtítulo e elemento opcional à direita e um slot padrão para conteúdo.
---
<br />

## Recomendamos o uso de PainelCards quando:
- Pretende-se utilizar um card que use esse mesmo padrão de estilo descrito.

<br />

## Não recomendamos o uso de PainelCards quando:
- For necessário customizar o header do card de forma diferente da descrição.
