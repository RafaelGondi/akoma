# SideSheet

### SideSheets são componentes usados para mostrar conteúdo em uma estrutura de modal, mas que é apresentado em um dos lados da página. São utilizados para sobrepor o conteúdo na interface com o objetivo de concentrar a atenção do usuário em informações específicas ou para realizar uma tarefa específica.
---
<br />


## Recomendamos o uso de SideSheets quando:
- For necessário exibir uma lista de ações que afetam o conteúdo principal da tela, como filtros.
- For necessário exibir conteúdo e recursos complementares aos da página.

<br />

## Não recomendamos o uso de SideSheets quando:
- For necessário o uso de um modal convencional.
- For necessária uma solução de navegação.
- For necessário um form para registrar algo.

<br />

## Observações:
- O SideSheet fornece conteúdo contextualmente relevante para a visualização. Este componente foi projetado apenas para usos em desktop e tablet.
