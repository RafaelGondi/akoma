# SideSheet

### O SideSheet é um componente utilizado para exibir um conteúdo que contextualiza a tela principal em uma estrutura de modal, sendo apresentado em um dos lados da página. É utilizado para sobrepor o conteúdo na interface com o objetivo de concentrar a atenção do usuário em informações específicas ou para realizar uma tarefa específica.
### <strong>Observação:</strong> O SideSheet fornece conteúdo contextualmente relevante para a visualização. Este componente foi projetado apenas para experiências de médio a grande porte (Desktop e Tablet).
---
<br />


## Recomendamos o uso de SideSheets quando:
- For necessário exibir uma lista de ações que afetam o conteúdo principal da tela, como filtros.
- For necessário exibir conteúdo e recursos complementares.
- For necessário fornecer acesso rápido a informações detalhadas.

<br />

## Não recomendamos o uso de SideSheets quando:
- For necessário o uso de um modal convencional.
- For necessária uma solução de navegação.
- For necessário um form para registrar algo.
