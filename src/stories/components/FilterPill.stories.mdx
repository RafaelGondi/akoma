# FilterPill

### FilterPills são componentes utilizados para ativar popovers contendo filtros para listas e views.
---
<br />

## Recomendamos o uso de FilterPills quando:
- Você estiver trabalhando com filtros.
- Recomendamos seu uso em conjunto com popovers contendo opções de filtragem.

<br />

## Não recomendamos o uso de FilterPills quando:
- Você estiver procurando por um botão.
- Você estiver procurando um dropdown.
- Você estiver trabalhando com tabelas.
