# EmptyState

### EmptyStates são componentes utilizados em momentos instrutivos aos usuários

---
<br />

## Recomendamos o uso de EmptyStates quando:
- Para instruir o usuário sobre o funcionamento de uma determinada página.
- Uma página ou componente não possuir conteúdo padrão para ser mostrado.
- Uma determinada ação resultar em erro.

<br />

## Não recomendamos o uso de EmptyStates quando:
- Como um loader.
- Quando utilizado apenas com título.
