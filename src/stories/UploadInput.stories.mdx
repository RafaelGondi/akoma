# UploadInput

### UploadInput é um input customizado de arquivos.
---
<br />

## Recomendamos o uso de UploadInputs quando:
- Você precisar que o usuário faça uploads do seu computador para a plataforma.
- Você quiser prover uma experiência de arrasta e solta para upload de arquivos.

<br />

## Não recomendamos o uso de UploadInputs quando:
- A sua experiência for mobile-first
