# Timeline

### Timelines são componentes utilizados para apresentar informações em ordem cronológica.
---
<br />

## Recomendamos o uso de Timelines quando:
- Você tem uma série de informações que podem ser categorizadas por data.

<br />

## Não recomendamos o uso de Timelines quando:
- Você precisar de um componente de stepper.
- A ordem das informações que você está apresentando é irrelevante.

<br />
