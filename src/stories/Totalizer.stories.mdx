# Totalizer

### Totalizers são componentes utilizados para consolidar dados importantes e que devem ser acessados rapidamente.
---
<br />

## Recomendações de uso:
- Use o totalizer quando for preciso evidenciar dados importantes para acompanhamento do usuário.
- Utilize totalizers, preferencialmente com ícones, para ajudar na identificação da informação.
- Utilize totalizers preferencialemente com números.
- Utiliza o posicionamento do totalizer de maneira consistente na mesma view.

<br />

## O que evitar:
- Utilizar totalizers em locais pouco acessíveis aos usuários.
- Utilizar apenas texto nos totalizers.
- Utilizar totalizers com informações pouco relevantes no contexto de uso.
