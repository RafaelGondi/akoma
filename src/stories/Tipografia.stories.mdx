# Tipografia

A Tipografia padrão do Cuida e do Cidade Saudável é a Inter. A Inter é uma fonte especialmente projetada 
para interfaces de usuário e tem foco em alta legibilidade de textos de tamanho pequeno a médio em telas de computador.
A família apresenta uma boa “Altura-x”  (altura das letras minúsculas) o que auxilia na legibilidade de textos mistos e 
minúsculos. Os pesos de fonte utilizados são principalmente o Regular(400) e Medium (500), com alguns usos de Semi Bold (600).

<hr />
<br />

## Recomendações

- Para facilitar a leitura e deixar a interface mais leve é
recomendável que as cores para tipografia, quando em backgrounds claros,
sejam tons de azul, e nunca cores “chapadas”.

- Para melhor compreender a hierarquia das informações, é recomendável
a utilização de diferentes cores neutras representando hierarquias
distintas na escala tipográfica.

- Para facilitar o processo de escaneamento da interface, evitar utilizar títulos,
 lables, botões e demais elementos da interface com todas as letras maiúsculas, sendo
	recomendado esse recurso apenas para criar diferenças na hierarquia tipográfica.

- Para evitar problemas de legibilidade, não é recomendável a utilização de textos itálicos.

- Para evitar problemas de legibiliade e erros de interpretação da interface, recomendamos
 a utilização de textos sublinhados apenas quando necessário como recursos para realçar links.

- Para facilitar o escaneamento e trabalhar a hierarquia de conteúdo, recomendamos a 
utilização pontual de textos em negrito.
