# Iconografia

Simplifica a interação do usuário com o sistema, pois possibilita que elementos
sejam mais facilmente identificados. A iconografia permite que os usuários compreendam
com mais facilidade os componentes do layout, reduzindo a curva de aprendizado do usuário.

<hr />
<br />

## Recomendações

-  Para facilitar a identificação de elementos interativos, recomendamos que sejam utilizados ícones com bordas suavizadas e outlined.

-  Para melhor representar ícones do sistema, recomendamos que sejam utilizados ícones com tamanho 18 x 18 pixels.

-  Para trazer uma melhor experiência visual ao usuário, recomendamos que sejam utilizados ícones de forma consistente (ex.: utilizar sempre ícones da mesma família).

- Para melhor compreensão por parte do usuário, utilize ícones já estabelecidos para a função desejada

- Para representar variações de um mesmo conceito, recomendamos o uso de apenas um ícone.

- Para tornar o conteúdo mais claro, utilize ícones em conjunto com texto, sempre que possível.

## Ícones

Os ícones utilizados são do conjunto <a href="http://ionicons.com/">Ionicons</a>. Para informações detalhadas sobre a utilização do pacote, recomendamos a leitura da documentação.
